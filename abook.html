abook - carnet d'adresses

update27 mars 2016

abook est un gestionnaire de carnet d'adresses. Doté d'une interface en mode texte, il s'exécute dans un terminal ou une console. D'une utilisation simple, il est personnalisable via son fichier de configuration : ~/.abook/abookrc (champs, vues, couleurs, etc). Conçu pour fonctionner avec le client mail mutt, il permet d'exporter son ou ses carnets dans différents formats.
